Theming

The Studi√∂ Web Components come with styling for a light and a dark theme, as well as for scales of 100% and 125%.
To apply a light or dark theme, set either the studio-theme-light or the studio-theme-dark class on the <body> tag.
To apply a 100% or 125% scale, set either the studio-scale-100 or the studio-scale-125 class on the <body> tag.

Required CSS files

Theming uses the studio-light.css, studio-dark.css, studio-scale-100.css and studio-scale-125.css files. You should include at least one theme and one scale in your project. See the Getting started section for how to include the files with the CDN and NPM distributions.

Theming helper

You may want to use the Theming helper class to easily set the theme and scale on the <body> tag.
