Configuring the React application

This chapter comes with a full React project in the Studiö integration code base. The sample project uses TypeScript but the concepts are the same in other languages. The main steps in configuring your React application to use the Studiö Web Components are explained here.
If you are new to React, please refer to the React documentation.

Install the Studiö Web Components

Use npm install to install the Studiö Web Components in your project.
  • npm install

Install the Lit React wrappers

Use npm install to install @lit-labs/react in your project. More on this here.
  • npm install
This is required at the moment because React is not yet fully compliant with the Web Components standards. See here for the current score sheet.

Custom elements

In order to integrate the Studiö custom element in your React HTML templates, import the wrappers as follows from studio-react.
  • App.tsx
For instance, we will be using the <StudioSlider> React wrapper in the HTML rather than the <studio-slider> custom element. The syntax is different but the semantics are preserved.
The Studiö React wrappers are built on top of @lit-labs/react.

Fonts

Studiö uses the Bai Jamjuree font by default but you need to import it first in your index.css file. See the Styling chapter if you intend to use a different font.
  • index.css

Studiö styles

Impor the Studiö css files you need in your index.css file.
  • index.css
You many not need all of the above but should at least include one of either studio-scale-100.css or studio-scale-125.css, as well as one of either studio-light.css or studio-dark.css, based on what your application's requirements.
