Integration with React

The Studiö Web Components are easy to integrate with React applications.
Because Web Components are UI framework and library agnostic, the task of integrating with most UI frameworks and libraries is straightforward, but in the specific case of React you need to use the provided wrappers for React.
The reason you need to use wrappers for React has to do with how React is implemented. See Custom Elements Everywhere for a list of UI frameworks and libraries, including their level of compliance to the Web Components standards.
Until the issue is solved in React, Studiö includes React wrappers for consumption of Studiö elements from React applications.
