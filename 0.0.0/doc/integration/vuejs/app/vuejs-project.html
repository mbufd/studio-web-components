Configuring the Vue.js application

This chapter comes with a full Vue.js project in the Studiö integration code base. The main steps in configuring your Vue.js application to use the Studiö Web Components are explained here.
If you are new to Vue.js, please refer to the Vue.js documentation.
In this chapter we assume usage of Vue 3 with TypeScript.

Install the Studiö Web Components

Use npm install to install the Studiö Web Components in your project.
  • npm install

Custom elements schema

Indicate which tags are custom elements in your build. This is needed for any Web Component custom element and does not only apply to Studiö.
Please refer to Using Custom Elements in Vue for details. Here we demonstrate setting the options in the vue.config.js file.
  • vue.config.js

Importing the components

Add an import of the Studiö Web Components in your App.vue.
  • App.vue

Fonts

Studiö uses the Bai Jamjuree font by default but you need to import it first in the unscoped <style> section of your App.vue file. See the Styling chapter if you intend to use a different font.
  • App.vue

Importing the Studiö styles

Import the Studiö css files in your App.vue.
  • App.vue
You many not need all of the above but should at least include one of either studio-scale-100.css or studio-scale-125.css, as well as one of either studio-light.css or studio-dark.css, based on what your application's requirements.
