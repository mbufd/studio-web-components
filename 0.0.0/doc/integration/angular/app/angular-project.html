Configuring the Angular application

This chapter comes with a full Angular project in the Studiö integration code base. The main steps in configuring your Angular application to use the Studiö Web Components are explained here.
If you are new to Angular, please refer to the Angular documentation.

Install the Studiö Web Components

Use npm install to install the Studiö Web Components in your project.
  • npm install

Custom elements schema

Use CUSTOM_ELEMENTS_SCHEMA in your app.module.ts. This is needed in order to use any Web Component custom element in Angular and does not only apply to Studiö.
  • app.module.ts

Importing the components

Add an import of the Studiö Web Components in your app.module.ts.
  • app.module.ts

Fonts

Studiö uses the Bai Jamjuree font by default but you need to import it first in your styles.css file. See the Styling chapter if you intend to use a different font.
  • styles.css

Bundling the Studiö styles

Modify the styles property under the architect.build section of your angular.json file to add the Studiö css files you need.
If you omit the bundleName attribute, then the Studiö styles will be bundled together with the default styles.css bundle.
  • angular.json
