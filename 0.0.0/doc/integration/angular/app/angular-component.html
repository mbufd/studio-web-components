Selecting a theme and scale

To apply a light or dark theme, set either the studio-theme-light or the studio-theme-dark class on the <body> tag.
To apply a 100% or 125% scale, set either the studio-scale-100 or the studio-scale-125 class on the <body> tag.
If you need to toggle theme or scale in your application, you may want to use the Theming helper class.
