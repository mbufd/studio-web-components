Tick marks

The number and sizes of major and minor tick marks of the axis are configurable.

Number of sections

Use the major-sections attribute or majorSections property to configure the number of major sections.
The default is 10 major sections, which defines 11 major tick marks on the axis.

Use the minor-sections attribute or minorSections property to configure the number of minor sections.
The default is 5 minor sections, which produces 4 minor minor tick marks per majer section on the axis.

Tick mark selection

Use the ticks attribute to select which tick marks to display. Combine the l, r and c flags to select placement of tick marks on the left, right or center.
The default is lr, which selects both the left and right sides.
For an horizontal axis, l selects the top side and r selects the bottom side.

Tick mark length

Use the major-length and the minor-length attributes to set the length of the major and minor tick marks respectively.
Tick mark lengths are specified in units of px or %. The default is px if no unit is specified.
A % is relative to the thickness of the axis.
Use the major-length atttribute or the majorLength property to set the length of the major tick marks. The default is 33%.

Tick mark weight

Use the major-weight and the minor-weight attributes to set the weight in pixels of the major and minor tick marks respectively.
Use the major-weight attribute or the majorWeight property to set the weight of the major tick marks. The default is 1.

Rounded tick marks

Use the rounded-ticks to apply a rounded style to the major and minor tick ends. It is more noticeable when combined with a weight.
