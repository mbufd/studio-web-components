Attributes and properties

All attributes have a corresponding property named using the camel case convention.
fit
Specify the fit attribute or set the fit property to true as a shortcut to apply style="width: 100%; height: 100%" to the axis. This is useful when composing with some other components.
major-sections (majorSections)
Use the major-sections attribute or majorSections property to set the number of major sections.
The default is 10.
major-weight (majorWeight)
Use the major-weight attribute or majorWeight property to set the weight of major tick marks in pixels.
The default is 1.
major-length (majorLength)
Use the major-length attribute or majorLength property to set the length of major tick marks in px or % units.
The default is 33%.
The % unit is relative to thickness of the axis.
minor-sections (minorSections)
Use the minor-sections attribute or minorSections property to set the number of minor sections.
The default is 5.
minor-weight (minorWeight)
Use the minor-weight attribute or minorWeight property to set the weight of minor tick marks in pixels.
The default is 1.
minor-length (minorLength)
Use the minor-length attribute or minorLength property to set the length of minor tick marks in px or % units.
The default is 20%.
The % unit is relative to thickness of the axis.
orientation
Use the orientation attribute or property to configure the orientation of the axis.
Use h for horizontal, use v for vertical.
The default vertical.
rounded-ticks
Specify the rounded-ticks attribute or set the roundedTicks property to true to apply rounded tips to all tick marks.
ticks
Use the ticks attribute or property to select the location of ticks using the l, r and c flags, which can be combined.
The default is lr.

CSS custom properties

--studio-length
The length of the axis. This corresponds to the width if the axis is horizontal, or to the height if it is vertical. See the orientation attribute.
--studio-thickness
The thickness of the axis. This corresponds to the height if the axis is horizontal, or to the width if it is vertical. See the orientation attribute.
--studio-surface-color
The surface color for the axis.
--studio-content-color
The major tick marks color for the axis.
--studio-accent-color
The minor tick marks color for the axis.
