Size

The length and thickness of the axis are configurable using CSS custom properties.

Length

Use the --studio-length CSS custom property to configure the length of the axis.
The length corresponds to the height of a vertical axis or to the width of an horizontal axis.
The default is --studio-size-5b.

  • CSS

Thickness

Use the --studio-thickness CSS custom property to configure the thickness of the axis.
The thickness corresponds to the width of a vertical axis or to the height of an horizontal axis.
The default is --studio-size-1b.

Fitting inside of a container

Use the fit attribute or property to style the width and height of the axis to 100%, in order to fit a container.
This is a useful shortcut to fit a axis inside a <studio-slider> or inside a <studio-pad>.
