Styling with parts

If styling with the provided CSS custom properties is not enough, you can use one of the available slider parts to expand the styling capabilities.

The inner-surface part

Use the inner-surface part to style the main area of the slider.

The label parts

Use the label-start and label-end parts to style the ends of the slider.

The accent parts

Use the accent-tl and accent-br parts to style sides of the slider. The accent-tl part matches the top border of an horizontal slider or the left border of a vertical slider. The accent-br part matches the other side.

The cursor parts

Use the cursor part to style the cursor.

The cursor-decoration part

Use the cursor-decoration part to style the decoration inside the cursor.

The cursor range part

Use the cursor-range part to style the cursor range area.
