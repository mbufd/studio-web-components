Slider action

Drag the cursor to move it across the surface.
Alternatively, press on the slider surface then drag the cursor.
When pressing on either end of the surface, the slider will move to within a margin of the edge of the surface.

  • HTML
To limit the action to dragging the cursor only, use the no-direct attribute or set the noDirect property to false. This disables slider moves upon pressing on the surface.
