Sizes

Use the --studio-length and --studio-thickness CSS custom properties to set the length and thickness of the slider. You can also modify the size of the cursor and the size of the sides of the slider, as described below.

Length

Use the --studio-length to modify the length of the slider. The default length is --studio-size-5b.

The minimum length for a slider is --studio-size-2b.

Thickness

Use the --studio-thickness to modify the thickness of the slider. The default thickness is --studio-size-1b.
  • Example
  • HTML
  • Thickness 1b Thickness 2b Thickness 7u
    Thickness 2b
The minimum thickness for a slider is --studio-size-4u.

Cursor

Use the --studio-cursor-size CSS custom property to modify the size of the cursor along the length of the slider.
The default cursor size is --studio-size-1b
The minimum cursor size is --studio-size-4u and the maxium size is a third of the slider's length.
A longer cursor has less dynamic range.
