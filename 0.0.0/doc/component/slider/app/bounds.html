Bounds

Use the start-value attribute or startValueproperty to set the value bound for the bottom of a vertical slider or for the left end of an horizontal slider.
Use the end-value attribute or endValueproperty to set the value bound for the top of a vertical slider or for the right end of an horizontal slider.

Invert the bounds if needed, by swaping the low and high values. If you set start-value higher than end-value, then a vertical slider will display its lowest value at the top and an horizontal slider will display its lowest value on the right.

Changes to the bounds don't affect the cursor position but are reflected in the value. Setting the bounds to equal values will disable the slider and lock the cursor in place.
