Colors

Surface color

Use the --studio-surface-color CSS custom property to set the color of a slider.
  • Example
  • HTML

Inner surface color

Use the --studio-inner-surface-color CSS custom property to set the color of the inner surface. The ends of the cursor bump into the limits of the inner surface.
  • Example
  • HTML

Cursor range color

Use the --studio-cursor-range-color CSS custom property to set the color of the cursor range. The center of the cursor bumps into the limits of the inner surface.
  • Example
  • HTML

Cursor color

Use the --studio-cursor-color CSS custom property to set the cursor color of a slider.
  • Example
  • HTML

Accent color

Use the --studio-accent-color CSS custom property to set the accent color for the sides of a slider.
By default the cursor color is the same as the accent color. See above for how to modify the cursor color.
  • Example
  • HTML

Uniform color

Use the uniform attribute or property to set the color of the slider to the uniform color while retaining the surface color for interactions.
This works great in designs where you want a uniform color across a set of sliders while showing specific colors for specific sliders during interactions.
