Events

A <studio-slider> element generates the following events.
studio-slider-value when the value of the slider changes.
studio-slider-bounds when one of the bounds changes.
studio-slider-state when transiting between down and up state on interactions.
Each of the above events contains the studio member containing properties that are specific to the slider's event.

studio-slider-value event

Listen to the studio-slider-value event to watch for changes to the slider's value. On the event, the studio.value property will contain value of the slider.

studio-slider-bounds event

Listen to the studio-slider-bounds event to watch for changes to the slider's bounds. On the event, the studio.startValue and studio.endValue properties will contain the bounds for the slider's value.

studio-slider-state event

Listen to the studio-slider-state event to watch for changes to the slider's state. On the event, the studio.down property will be true or false depending on whether the mouse (or touch) is down or not.
