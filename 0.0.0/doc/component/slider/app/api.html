Attributes and properties

All attributes have a corresponding property named using the camel case convention.
animation
Use the animation attribute or property to set the animation speed when no-direct (noDirect) is set.
The value is in milliseconds. A value of 0 or less disables the animation.
disabled
Use the disabled attribute or property to disable interactions with the slider.
end-value (endValue)
Use the end-value attribute or the endValue property to access the end value of the slider.
The end value is the top bound of a vertical slider or the right bound of an horizontal slider.
no-direct (noDirect)
Use the no-direct attribute or the noDirect property to disable direct setting of the slider by pressing on the slider's surface.
orientation
Use the orientation attribute or property to configure the orientation of the slider.
Use h for horizontal, use v for vertical.
The default vertical.
start-value (startValue)
Use the start-value attribute or the startValue property to access the start value of the slider.
The start value is the bottom bound of a vertical slider or the left bound of an horizontal slider.
step
Use the step attribute or property to access the quantization increment. The slider's value will be a multiple of step unless it is set to 0, in which case no quantization occurs.
Default is 0.
uniform
Use the uniform attribute or property to use --studio-uniform-color as the color of the surface at rest. The color of the slider remains as specified using the --studio-surface-color when the slider is interacted with.
value
Use the value attribute or property to access the value of the slider.
The value is between start-value (startValue) and end-value (startValue).
The default value is 0.

CSS custom properties

--studio-length
The length of the slider. This corresponds to the width if the label is horizontal, or to the height if it is vertical. See the orientation attribute.
--studio-thickness
The thickness of the slider. This corresponds to the height if the label is horizontal, or to the width if it is horizontal. See the orientation attribute.
--studio-cursor-size
The size of the slider's cursor, on the length axis of the slider. Default is --studio-size-1b.
--studio-border-size
The size of the sides of the slider. Default is --studio-size-1u.
--studio-font-size
The font size for content of the slider. Default is --studio-size-font-xs.
--studio-surface-color
The surface color.
--studio-inner-surface-color
The inner surface color. The inner surface is inside the surface, within margin.
--studio-cursor-range-color
The color for the cursor range area. The cursor rage is inside the inside surface, but only extends to where the center of the cursor extends.
--studio-accent-color
The accent color. A single color applies to both sides of the slider.
--studio-uniform-color
The uniform color to use when applying the uniform attribute.
--studio-content-color
The content (text) color for the slider.

Parts

inner-surface
Use this part to style the main area inside the label.
label-start
Use this part to style label area at the bottom of a vertical slider or on the left of an horizontal slider.
label-end
Use this part to style label area at the top of a vertical slider or on the right of an horizontal slider.
accent-tl
Use this part to style side area on the left of a vertical slider or at the top of an horizontal slider.
accent-br
Use this part to style side area on the right of a vertical slider or at the bottom of an horizontal slider.
cursor
Use this part to style the slider's cursor.
cursor-decoration
Use this part to style decoration inside the slider's cursor.

Slots

Default slot
Content in the default slot will display in the main area (inner surface).
label-start
Use this slot to place content in the label area at the bottom of a vertical slider or on the left of an horizontal slider.
label-end
Use this slot to place content in the label area at the top of a vertical slider or on the right of an horizontal slider.

Events

studio-slider-value
Generated when the value of the slider changes.
studio-slider-state
Generated when the up or down state of the slider changes.
studio-slider-bounds
Generated when the bounds of the slider change.

Methods

No special method.
