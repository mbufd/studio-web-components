Attributes and properties

All attributes have a corresponding property named using the camel case convention.
bold
Use the bold attribute or property to turn the text in the default slot to bold.
inset-label (insetLabel)
Use the inset-label attribute or the insetLabel property to move all labels within a margin of the outer surface.
selected
Use the selected attribute or property to highlight the pad as selected.
uniform
Use the uniform attribute or property to use --studio-uniform-color as the color of the surface at rest. The color of the pad remains as specified using the --studio-surface-color when the pad is hovered, selected or active.
swipe
Use the swipe attribute or property to enable swiping on the pad. This will generate studio-pad-swipe events when the user swipes on the pad.
xy-fx (xyFx)
Use the xy-fx attribute or the xfFx property to configure a visual effect that will show when the user clicks or swipes on a pad.
Use xy to enable the effect in both the X and Y directions, use x to enable only on X or use y to enable only on Y.
When the swipe attribute is present, the effects will follow the swipe movement. Otherwise it the effect will stand still at the point where the click occured.

CSS custom properties

--studio-content-color
The color of the text content for both the default and the secondary slots.
--studio-fx-color
The color of the XY effects when they are enabled using the xy-fx attribute or the xyFx property.
--studio-height
The height of the pad.
--studio-highlight-color
The color of the outer surface highlight when the pad is interacted with.
--studio-selected-highlight-color
The color of the inner surface highlight when the pad has the selected attribute.
--studio-surface-color
The color of the surface, applicable to both the outer and the inner surface.
--studio-uniform-color
The uniform color to use when applying the uniform attribute.
--studio-width
The width of the pad.

Parts

The surface of a part is split in two layers: the outer-surface and the inner-surface. The inner surface is a layer above the outer surface and is smaller by a margin. Both surfaces are visibile when making the inner surface translucent. The outer surface acts as a border when the inner surface is opaque.
outer-surface
Use this part to style the outer surface of the label.
Using a border on the outer surface will alter the size of the pad and affect layouts when using block units.
inner-surface
Use this part to style the surface of the pad.

Slots

Default slot
Content in the default slot will display on the inner-surface. Text will wrap if too long.
Secondary slot
Content in the secondary slot will display on the inner-surface, below the content of the default slot.

Events

All pad events properties are as follows:
  • Event properties
studio-pad-down
Generated at the moment the pad is clicked or touched. The state becomes down until a studio-pad-up occurs.
studio-pad-move
Generated when a movement is detected over the pad while the state is not down.
studio-pad-up
Generated when the pad click or touch is released.
studio-pad-swipe
Generated when a movement is detected over the pad while the state is down.

Methods
