A toy drum machine

This is example shows a toy drum machine with 8-step sequencer and 16 drum pads.
Step editing
  • Select EDIT.
  • Select a sequencer step (1 through 8).
  • Toggle the drum pads for that step.
  • Repeat the above for other steps.
  • (optional) Use RESET to start edit over from a blank sequence.
Playback
  • After step editing, select PLAY.
  • Hit or swipe a drum pad to play sounds manually.
  • Swiping vertically changes the volume.
  • Swiping horizontally changes the filter.
  • Use STOP or EDIT to end playback.
