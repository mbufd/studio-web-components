Events

A <studio-pad> element generates the following events.
studio-pad-down indicates that the pad transited to the down state. This event occurs immediately as the pad is clicked or touched. There is no pad click event as such.
studio-pad-up indicates that the pad transited back to the up state.
studio-pad-swipe indicates a swiping motion while in down state.
studio-pad-move indicates a mouse move (hover) while in up state.

Event properties

Each of the above events contains the studio member containing the offset and pointerEvent properties.
The offset property contains the x and y coordinates for the event, as values normalized between 0 and 1.
