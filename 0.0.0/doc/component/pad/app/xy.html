XY sensitivity

A pad comes with XY sensivity to mouse or touch.

Swiping

The presence of the swipe attribute enables swiping on the pad.
See the Events section below for events generated while swiping.
Swiping will provide visual feedback if the xy-fx attribute is set to xy, x or y.
