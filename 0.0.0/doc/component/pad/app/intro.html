The <studio-pad> component renders a control pad with X-Y sensitivity. For instance, when used to control a sound, the X axis may control a filter where the Y axis may control the volume level.
In it's simplest configuration, a pad acts as a button and can show a selected state, to indicate for instance that the pad is in action. A pad can be decorated with up to 4 labels and each side. On touch-capable devices, you can control multiple pads simulataneously using mutiple touches or swipes.
