Styling using parts

If styling with the provided CSS custom properties is not enough, you can use the outer-surface and inner-surface parts to expand the styling capabilities.
The surface is composed of an outer part as well as an inner part. The outer part looks like a border when the inner part is opaque, and will show through if the inner part is made translucent. By default, both parts are of color --studio-surface-color and the inner part increases in brightness when hovering or activating the pad.

Outer surface part

The outer-surface part provides the background and border for the pad.
