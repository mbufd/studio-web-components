Intensity

You can adjust the intensity of the LED to have the light shine more or less when the LED is on.
Use the normalized-intensity attribute or the normalizedIntensity proprety to set the intensity as a normalized value between 0 and 1, corresponding to an intensity range that you set using the min-intensity and max-intensity attributes. Default is .5, meaning at half point in the intensity range.
Shown here are intensities from 0 to 1 in steps of 0.25, given the default intensity range of 100 to 150.
