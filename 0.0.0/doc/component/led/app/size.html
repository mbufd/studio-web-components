Sizes

The <studio-led> component can be sized using the --studio-width and --studio-height CSS custom properties. The default value is var(--studio-size-3u) in both dimensions.
You may specify any size or one of the curated sizes, such as var(--studio-size-2u), var(--studio-size-3u) or var(--studio-size-4u). See about sizes under the Styling chapter for more information.
