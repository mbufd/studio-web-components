Border color

You can set the border color of the LED using the --studio-border-color CSS custom property, either from your .css file or inline with the style attribute.

Border width

You can set the border width of the LED using the --studio-border-width CSS custom property, either from your .css file or inline with the style attribute.
The border width expands inward such that the size of the LED is kept constant (unless you make the width too large).
