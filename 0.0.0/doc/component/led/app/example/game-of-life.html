John Conway's game of life

This example shows a set of studio-led arranged in a matrix serving as a display for John Conway's game of life. Played here on a finite, toroidal space.

Click on the board to interfere by adding a blinker pattern.
Play again
LED intensity is varied using the normalizedIntensity property to show how living cells age. The maximum LED intensity is set to 220 (%) using the maxIntensity property so to reach a bright yellow once stable for a while. The LED border is made transparent using the --studio-border-color CSS custom property.
