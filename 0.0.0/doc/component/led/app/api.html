Attributes and properties

All attributes have a corresponding property named using the camel case convention.
on
The on boolean attribute and property are used to access the on or off state of the LED.
See normalized-intensity below for how you can access the intensity of the LED when it is on.
clikable
The clikable boolean attribute and property are used to access the clickable state of the LED.
It is not set by default. When set, the LED will emit standard onclick events.
normalized-intensity (normalizedIntensity)
Use the normalized-intensity attribute to access the normalized intensity of the LED from 0 to 1.
A normalized intensity of 0 matches the value of min-intensity, while an intensity of 1 matches the value of max-intensity.
Intensity is ignored when the LED is off.
min-intensity (minIntensity)
Use the min-intensity attribute to access the minimum intensity of the LED when it is on.
The value is an intensity ratio expressed as a percentage of lightness relative to the color indicated by the --studio-surface-color CSS custom property. Default is 100. The value can be set to higher than 100 in order to brighten up the LED.
max-intensity (maxIntensity)
Use the max-intensity attribute to access the maximum intensity of the LED when it is on.
The value is an intensity ratio expressed as a percentage of lightness relative to the color indicated by the --studio-surface-color CSS custom property. Default is 150. The value can be set to higher than 100 in order to brighten up the LED.
corners
Use the corners attribute to access the type of corners for the LED.
Allowed values are rounded and angled. Default is rounded.

CSS custom properties

--studio-width
The LED's width. Default is var(--studio-size-3u).
--studio-height
The LED's height. Default is the specified width.
--studio-led-off-opacity
The LED opacity when it is fully off. Must be a value between 0 and 1. Default is 0.2.
--studio-border-color
The border color for the LED.
--studio-border-width
The thickness of the LED's border. This does not change the size of the LED because the border expands inward rather than outward.
--studio-surface-color
The surface color for the LED.

Parts

surface
Use this part to style the surface of the LED.
border
Use this part to style the border of the LED.

Slots

Default slot
Content in the default slot will display on top of the surface.

Events

No special event.
