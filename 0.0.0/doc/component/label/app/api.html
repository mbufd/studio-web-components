Attributes and properties

All attributes have a corresponding property named using the camel case convention.
accent
Use the accent attribute or property to configure if and how to display an accent color.
Use s to enable an accent only at the start of the label, use e to enable an accent only at the end of the label, or use se to enable accents at both ends of the label.
The default is no accent.
See the --studio-accent-color CSS custom property for setting the color for the accent.
orientation
Use the orientation attribute or property to configure the orientation of the label.
Use h for horizontal, use 90l for sideways as rotated 90° left from the horizontal, or use 90r for sideways as rotated 90° rigth from the horizontal .
The default horizontal.

CSS custom properties

--studio-length
The length of the label. This corresponds to the width if the label is horizontal, or to the height if it is sideways. See the orientation attribute.
--studio-thickness
The thickness of the label. This corresponds to the height if the label is horizontal, or to the width if it is sideways. See the orientation attribute.
--studio-surface-color
The surface color for the label.
--studio-content-color
The content (text) color for the label.
--studio-accent-color
The accent color for the label. A single color applies to both the start and end accents. See the accent attribute for how to enable accents.
--studio-label-accent-length
The length of the accent for the label. A single length applies to both the start and end accents. See the accent attribute for how to enable accents.

Parts

surface
Use this part to style the surface of the label.

Slots

Default slot
Content in the default slot will display on top of the surface.
