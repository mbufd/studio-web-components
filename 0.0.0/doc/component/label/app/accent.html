Accent

You can add an accent color to the start or to the end of a label, or both. The accent does not use extra space, the label size remains the same.
  • Example
  • HTML
  • s e se
By default the accent is --studio-tangerine, but you can assing a color using the --studio-accent-color CSS custom property. The color applies equally to the start and end.
  • Example
  • HTML
  • mint pink turquoise
By default the accent of size 1u, but you can assing a length using the --studio-label-accent-length CSS custom property. The length applies equally to the start and end.
  • Example
  • HTML
  • 2u 3u 4u
Accents apply the same sideways.
s - left e - left se - left s - right e - right se - right
