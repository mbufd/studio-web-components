Use the NPM distribution to integrate and bundle the components with your applications. Please refer to the integration chapters for Angular, Vue.js or React.
This chapter shows how to include the CDN distribution using link and script tags, as well as the basics about how to use the components from JavaScript.

Browser support

Both distributions are available as ES Modules (JavaScript modules) for use in evergreen modern Web browsers only.
Recent versions of Chrome, Safari, Firefox, chromium based Edge and Samsung Internet browser are supported.
