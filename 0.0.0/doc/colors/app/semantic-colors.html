Semantic colors

Studi√∂ defines a small set of semantic colors that are designed to work well across paper colors and themes. The CSS custom properties for these colors are --studio-semantic-alert, --studio-semantic-warning, --studio-semantic-good and --studio-semantic-action.
The semantic colors are meant to work well when writing with the ink color --studio-semantic-color-ink which is white.
There are color name aliases for each of the semantic colors: --studio-semantic-red, --studio-semantic-orange, --studio-semantic-green and --studio-semantic-blue.
