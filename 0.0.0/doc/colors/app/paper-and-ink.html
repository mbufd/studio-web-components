Paper and ink

Text can be written with ink on paper. The paper and ink CSS custom properties are meant for general use where text is written on a page or within a section that acts as a page. The StudiÃ¶ built-in themes are designed to provide neutral shades of ink and paper. The dark theme (studio-dark.css) has paper in dark shades of gray, while the light theme (studio-light.css) has paper in light shades of gray. Then, the inks have opposite shades because they are intended for use on paper.

Action inks

In addition to the standard inks, there is an additional set of inks to use for use when writing text that is meant to convey an action, for instance for use with <a> tags. These inks are named --studio-action-ink-10 to --studio-action-ink-50 for use with paper of the same number.
