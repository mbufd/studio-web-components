Hues

Studiö comes with a selection of 16 hues from which curated colors sets are built, and from which you can build custom shades using saturation and lightness.

Custom colors based on standard hues

To make a color based on a hue, use the hsl CSS function with one of the Studiö hue CSS proprties, such as --studio-hue-red. Substitute the hue names to get different hues.

Tinted paper

You can create tinted paper using the hue with the hsl CSS function by setting the saturation to --studio-paper-tint-s and the lightness to --studio-paper-10-tint-l or any of the other paper shades --studio-paper-20-tint-l, --studio-paper-30-tint-l --studio-paper-40-tint-l or --studio-paper-50-tint-l. Regular paper inks will work fine on tinted paper.
