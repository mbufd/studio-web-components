Color sets

Studiö provides a set of curated colors for use in components as well as for general use in applications, for instance when applying colors to graphics.
Each color set is based on the 16 Studiö hues.
Color sets are intended for decoration of the Studiö components and are not meant to carry semantics. When using contrasting shades for different component parts, color-challenged people should be able to use the decorated controls without needing to make colors apart. The concept is very much like what you would find on a piece of hardware where controls are decorated with colors.

Basic colors

The basic color set contains 16 pure colors plus 16 matching desaturated colors. Simply prefix --studio- to any the 32 basic color names from the above basic color set, for instance --studio-turquoise. For text over these colors, we provide --studio-ink-light, which is a white ink (alghough this can be challenging over the lemon or lime colors).

Flower colors

The flower set is made of light, pastel colors. Use the prefix --studio- and the sufffix -flower with any the 16 hue names, for instance --studio-turqoise-flower. Unlike the other color sets, it requires that text be written in a dark color. For this purpose, we recommend using --studio-ink-flower which is a near-black shade.

Night colors

The night set is a collection of tinted blacks. Use the prefix --studio- and the sufffix -night with any the 16 hue names, for instance --studio-turquoise-night. Use --studio-ink-night for text over these colors.
