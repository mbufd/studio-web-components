Theming

To apply a 100% or 125% scale, set either the studio-scale-100 or the studio-scale-125 class on the <body> tag.
See the Theming chapter for details.

Scaling

Scaling is done automatically for any of the --studio-size-... CSS custom properties.
However, when using custom sizes in px units, use the --studio-scale CSS custom property along with the calc function to apply scaling according to the current scale.
