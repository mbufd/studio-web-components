Block unit and sizes

Studiö includes a block unit called b.
Elements can be of any size but Studiö comes with standard block sizes as well as a standard gap size to space blocks apart. This enables easy standardization of element sizes in an application.

Looking at the chart above, you find that a block size of --studio-size-2b is twice the block size --studio-size-1b, plus one standard gap size --studio-size-gap.
This is because a multiple of the gap size is accounted for in every block size greather than 1b.

Formula for block sizes

The formula for a block of size is:
1b = 8 * 1u
Nb = {N}*1b + {N-1}*gap
where N > 1 and gap is the standard gap size (1u).
