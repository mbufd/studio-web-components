Sizes for layout

Gap size

Layout CSS classes use a fixed gap size of --studio-size-layout-gap to space elements from each other.
By default --studio-size-layout-gap equals the default gap size defined as --studio-size-gap, such that gaps will add up conveniently when using block sizes as described below.
Note that you can customize --studio-size-layout-gap if needed for a custom layout, but you should leave --studio-size-gap untouched.
